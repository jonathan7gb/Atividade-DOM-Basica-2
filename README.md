2 - Atividade: Manipulador de Cores e Fontes

Objetivo: Nesta atividade, vamos utilizar o JavaScript e a Manipulação de DOM para criar uma página simples que será capaz de alterar dinamicamente a cor de fundo, a cor do texto e a fonte do texto em um elemento HTML.

Requisitos:

A página deve conter um título (elemento h1) e um parágrafo (elemento p).
A página deve conter botões com as seguintes funcionalidades:
"Alterar Cor de Fundo": Ao ser pressionado, deve alterar a cor de fundo da página para uma cor RGB aleatória.
"Alterar Cor do Título": Ao ser pressionado, deve alterar a cor do texto do título para uma cor RGB aleatória.
"Alterar Cor do Parágrafo": Ao ser pressionado, deve alterar a cor do texto do parágrafo para uma cor RGB aleatória.
"Alterar Fonte do Título": Ao ser pressionado, deve alterar a fonte do texto do título para uma fonte selecionada aleatoriamente de uma lista de 15 fontes do Google Fonts.
"Alterar Fonte do Parágrafo": Ao ser pressionado, deve alterar a fonte do texto do parágrafo para uma fonte selecionada aleatoriamente de uma lista de 15 fontes do Google Fonts.
Deve haver também um quarto botão "Alterar Tudo", que ao ser pressionado, realiza todas as alterações anteriores de uma vez.
Dicas:

Para gerar cores RGB aleatórias, você pode utilizar a função Math.random do JavaScript.
